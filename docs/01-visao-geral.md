# Visao Geral

A plataforma e um SaaS simples para lojas de carros atenderem clientes pelo WhatsApp com apoio de um agente de IA.

O backend centraliza autenticacao, banco de dados, Evolution API, OpenAI, CRM mock, leads e handoff humano. O worker usa Redis para debounce e processamento assincrono. O CRM mock simula estoque e vendedores.
